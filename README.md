# MERN Music Player 🎵

A full stack music player built using the MERN stack.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Redux

## 🎥 Project Demo
[![Watch the demo](https://img.youtube.com/vi/DzMkFdpO6A8/0.jpg)](https://youtu.be/DzMkFdpO6A8)

## Features
- User authentication
- Play / Pause music
- Song browsing
- Playlist management

## Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
